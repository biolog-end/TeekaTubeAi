Bookmarklet для сбора ID с главной YouTube

Чтобы быстро собрать ID видео с главной страницы YouTube:

  1. Создайте новую закладку в вашем браузере.
  2. В поле "Адрес" или "URL" закладки вставьте весь код из поля ниже (начиная с `javascript:`).
  3. Назовите закладку, например, "Собрать ID YouTube".
  4. Перейдите на главную страницу youtube.com
  5. Нажмите на созданную закладку.
  6. Скопируйте ID из появившегося окна и вставьте в поле выше.
Внимание: Этот bookmarklet зависит от текущей структуры HTML-кода YouTube и может перестать работать, если Google ее изменит.
